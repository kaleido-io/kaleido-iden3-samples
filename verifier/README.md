# kaleido-iden3-verifier

Microservice for the Decentralized Identity verifier server based on iden3

## Build

```shell
make build
```

## Run

```shell
# Swagger: http://127.0.0.1:8000/api
# Boilerplate /status: http://127.0.0.1:8000/api/v1/status
make run
```

## Test

```shell
make test
```

## Docker

```shell
make docker
```
